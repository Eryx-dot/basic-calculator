# Calculator Web App

A simple calculator web application built with Python Flask.

## What is This?

This is a backend server that performs basic math operations:
- Addition
- Subtraction
- Multiplication
- Division (with zero-check)

## Technologies Used

- Python 3 - Programming language
- Flask - Web framework (backend)
- HTML - Frontend (interface)
- SQLite - (will be used in future projects)

## How to Run

### Step 1: Install Flask
```bash
pip install flask
```

### Step 2: Navigate to Project
```bash
cd calculator
```

### Step 3: Run the Server
```bash
python app.py
```

### Step 4: Open in Browser
Go to: `http://localhost:5000`

## Features

✅ Add two numbers  
✅ Subtract two numbers  
✅ Multiply two numbers  
✅ Divide two numbers (prevents division by zero)

## How to Use

1. Enter two numbers in the form
2. Select an operation from dropdown
3. Click "Calculate"
4. See the result
